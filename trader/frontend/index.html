Администрирование

Настройки брокера, капитала, уведомлений и системы
Брокер и API-подключение
Безопасность: токены хранятся локально в SQLite. Никогда не передавайте API-ключи третьим лицам. Для тестирования используйте ключи с правами только чтения.
URL брокера — адрес API сервера для подключения к торгам. Для БКС, Тинькофф, Финам, ВТБ URL подставляется автоматически — вручную менять не нужно. Поле актуально только при подключении нестандартного брокера. Настроить вручную
Управление капиталом робота
Рекомендация: создайте у брокера отдельный суб-счёт специально для робота. Это изолирует риски и позволяет точно контролировать P&L алгоритма. Укажите API-ключ именно этого субсчёта в настройках брокера выше.

Торговые параметры и риск-менеджмент
Уведомления — мессенджер MAX
Используется мессенджер MAX (VK). Создайте бота на business.max.ru → Чат-боты → Интеграция → Получить токен.
User ID — ваш числовой ID в MAX. Чтобы узнать его: напишите боту @getmyid_bot в MAX или возьмите число из ссылки на ваш профиль.
Система принимает форматы: 631107420238 или id631107420238 — число будет извлечено автоматически.
